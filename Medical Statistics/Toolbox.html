How do you know what package is right for you

Task view

CRAN groups all of its packages by their functionality/topic into 35 “themes.” It calls this its “Task view.” This at least allows you to narrow the packages you can look through to a topic relevant to your interests.

RDocumentation

There is a great website, RDocumentation, which is a search engine for packages and functions from CRAN, BioConductor, and GitHub (ie: the big three repositories). If you have a task in mind, this is a great way to search for specific packages to help you accomplish that task! It also has a “task” view like CRAN, that allows you to browse themes.

Update packages

help

If you still have questions about what functions within a package are right for you or how to use them, many packages include “vignettes.”

browseVignettes("package_name")
